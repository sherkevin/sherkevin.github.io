关于我

我是一个偏研究型的大模型应用算法候选人，主线是 calibration 与 reliability：多模态模型是否有视觉证据、 embedding 分数是否真的代表可替代性、会话推荐何时停止追问、领域 RAG 是否能有证据地回答并在高风险场景拒答。

我的经历覆盖 MiniMax 的多模态推荐与 LLM4Rec、字节跳动的 Tool Use / Agent 基础设施，以及 SMEE 的领域 LLM + RAG 工业诊断系统。当前目标是头部公司的 大模型应用算法正式岗位，兼顾工程落地与研究问题抽象。

研究方向

  • 多模态可靠性：幻觉校准、视觉 grounding、decoding-time verification。
  • RAG 与 Tool Use：证据检索、引用支持、schema grounding、安全工具调用。
  • LLM4Rec：多模态召回、Semantic ID、会话推荐、停止决策。
  • 后训练数据：SFT/DPO 数据构造、拒答行为、badcase 诊断、评测设计。

动态

  • 2026 面向大模型应用算法正式岗位求职中。
  • 2025 加入 SMEE 突破小组，参与领域 LLM + RAG 光刻机故障诊断系统。
  • 2025 在 MiniMax 实习，参与多模态召回、音色推荐与 LLM4Rec 评测。
  • 2025 在字节跳动实习，参与 Agent Infra、MCP Factory 与 Tool Use 数据基础设施。

代表研究

ACM MM under review · 独立一作

CHORD: Calibrating Hallucinations via Object-Resonant Decoding in MLLMs

面向多模态大模型 object hallucination 的 decoding-time 校准方法。 核心思想是结合 query-relevant visual anchors、late-layer visual attention 与 short-horizon rollout，检查候选生成路径是否保持视觉 grounding。

双盲审稿期间暂不公开全文，仅提供研究摘要。

RecSys under review · 独立一作

Does Similarity Imply Substitutability? Typed Score Corrections for Frozen Alignment

研究 frozen text-to-CF alignment 中的 relation collapse：语义相似不一定等于行为可替代。 通过 typed post-scoring correction 校准推荐候选角色。

RecSys under review · 独立一作

When to Stop Asking? A Top-K-Aware Stopping Layer for Conversational Recommendation

面向会话推荐的停止决策：不只看 top-1 confidence，而是根据 Top-K boundary stability 判断继续追问或直接推荐，平衡成功率与用户交互成本。
