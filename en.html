Worldview

Intelligence is compression; higher intelligence is reconstruction.

Compression makes the world representable. Decompression unfolds a representation. Reconstruction asks a system to recover structure from fragments, which reflects a deeper understanding of the world.

Memory, tools, and environment are different things.

LLM + memory resembles a brain; tools resemble a body; environment is the world. Memory stores reflection and compressed experience. Environment stores code, data, documents, collaboration context, and concrete artifacts.

Attention is the core of human-AI collaboration.

When building context, humans decide what matters and what can be ignored. This act of attention separates effective collaboration from blind automation.

Frequency shapes memory; value drives reflection.

Models learn general frequency patterns, while personal memory is shaped by the specific frequency of an individual's world. Valuable events trigger reflection, which is an iterative compression of high-value samples.

AI coding is entropy control and minimal change.

Code is an artifact deposited into the environment. Good AI coding is not large-scale generation; it is the smallest necessary change after understanding the project structure, keeping the system readable, maintainable, and reversible.
